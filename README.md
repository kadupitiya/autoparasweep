# Auto parameter sweeping
  
## Install and run

* docker build -t kadupitiya/para_sweep .
* docker run -itd -p 8181:8181 kadupitiya/para_sweep

## download and run

* docker pull kadupitiya/para_sweep
* docker run -itd -p 8181:8181 kadupitiya/para_sweep

## install and run in your linux computer

* make install
* make run
